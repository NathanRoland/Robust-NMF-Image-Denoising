# Robust-NMF-Image-Denoising
This report evaluates the robustness of Classical, L1-, and L2,1-norm NMF algorithms on noisy face image datasets (ORL and Extended YaleB). Using salt-and-pepper noise, models are compared via RRE, Accuracy, and NMI to determine which NMF variant best withstands high-magnitude corruption in real-world scenarios.
